# 🚗 Car Price Prediction Using Machine Learning

## 📌 Overview
This project predicts the price of used cars using Machine Learning. The dataset was cleaned, preprocessed, and used to train a Linear Regression model for price prediction.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## 📂 Dataset
- Quikr Used Car Dataset (`quikr_car.csv`)

## ✨ Features
- Data Cleaning
- Data Preprocessing
- Handling Missing Values
- One-Hot Encoding
- Linear Regression Model
- Car Price Prediction

## 📊 Model Performance
- **Algorithm:** Linear Regression
- **R² Score:** **0.19**

## 📁 Project Files
- `Car_prediction.ipynb` – Complete Google Colab notebook
- `quikr_car.csv` – Dataset
- `README.md` – Project documentation
